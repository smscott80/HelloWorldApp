# Hello World Windows Forms Application

A simple Windows Forms application that displays a "Hello World!" message when a button is clicked.

## Features

- Simple form with a button
- Displays a message box with "Hello World!" when button is clicked

## Requirements

- .NET 8.0 or later
- Windows OS (for running the application)

## Building

```bash
dotnet build
```

## Running

```bash
dotnet run
```

## Project Structure

- `Program.cs` - Entry point of the application
- `Form1.cs` - Main form with button and message box functionality
- `HelloWorldApp.csproj` - Project file
